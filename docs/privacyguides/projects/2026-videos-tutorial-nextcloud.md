Our video production team is primarily funded by general donations and monthly membership support. We have a large queue of videos that we want to produce, and a very small team working to make them happen, so we have to prioritize which videos we produce based on the resources we have available.

This is a lower priority video that we want to produce, but it is a very important topic that we want to cover. This video will be a tutorial on how to set up a Nextcloud Server, which is a tool that can help people host their own private cloud storage solution. We believe that providing high-quality, informative videos on important topics like this is crucial for raising awareness and empowering individuals to take action to protect their privacy rights.

Reaching this goal will allow us to prioritize the creation of this video, and ensure that we can produce it with the quality and attention to detail that it deserves.
